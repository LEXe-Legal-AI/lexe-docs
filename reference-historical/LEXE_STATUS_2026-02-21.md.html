LEXORC Pipeline Architecture

Multi-Agent Navigation & Understanding System — 6-phase agentic pipeline with 5 specialized agents, parallel execution, circuit breaker, and graceful LEGIS fallback chain.
PhaseAgentFunction
1. INTAKEClassifierComplexity classification: FLASH / STANDARD / DEEP / STRATEGIC
2. PLANNINGWorkPlanTask generation with Plan A/B for STRATEGIC queries
3. RESEARCHParallelEngine3 agents: NormAgent + CaseLawAgent + DoctrineAgent (asyncio.gather)
4. VERIFYAuditor (Censore)2-layer verification: L1 structural + L2 LLM coherence
5. SYNTHESIZESynthesizerVerified streaming output (sections A–F) + NBA generation
6. COMPLETE—Evidence persistence + Next Best Actions
